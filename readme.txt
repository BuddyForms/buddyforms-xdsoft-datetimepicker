== Description ==

This plugin integrate the DateTimePicker from XDSoft with BuddyForms.
Learn more about the component <a href="https://xdsoft.net/jqplugins/datetimepicker/" target="_blank">at!</a>
You need the BuddyForms plugin installed for the plugin to work. <a href="http://buddyforms.com" target="_blank">Get BuddyForms now!</a>

 == Installation ==

You can download and install BuddyForms XDSoft DateTimePicker by using the built in WordPress plugin installer. If you download BuddyForms XDSoft DateTimePicker manually, make sure it is uploaded to "/wp-content/plugins/".
